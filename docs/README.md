## 1. Цели и задачи тестирования

**Цель тестирования** — проверка корректности работы программных модулей, реализующих вычисление площади и периметра геометрических фигур.

**Задачи тестирования:**
- проверить правильность вычисления площади фигур;
- проверить правильность вычисления периметра фигур;
- выявить возможные ошибки при граничных значениях входных данных;
- автоматизировать процесс проверки с помощью unit-тестов.

---

## 2. Описание тестируемого продукта

Тестируемый продукт представляет собой набор Python-модулей, реализующих функции для вычисления площади и периметра следующих геометрических фигур:

- круг;
- прямоугольник;
- квадрат;
- треугольник.

Каждая фигура реализована в отдельном модуле и содержит функции для вычисления площади и периметра на основе входных параметров.

---

## 3. Область тестирования

В рамках данной лабораторной работы тестированию подлежат следующие компоненты:

- функция вычисления площади круга;
- функция вычисления периметра круга;
- функция вычисления площади прямоугольника;
- функция вычисления периметра прямоугольника;
- функция вычисления площади квадрата;
- функция вычисления периметра квадрата;
- функция вычисления площади треугольника;
- функция вычисления периметра треугольника.

Другие аспекты (пользовательский интерфейс, производительность, безопасность) в данной работе не рассматриваются.

---

## 4. Стратегия тестирования

Для тестирования используется **модульное тестирование (unit testing)**.

### Основные особенности стратегии:
- тестирование выполняется автоматически;
- каждый тест проверяет одну конкретную функцию;
- используются корректные и граничные входные данные;
- для проверки результатов применяются методы сравнения библиотеки `unittest`.

Тестирование реализовано с помощью стандартного модуля `unittest`, входящего в состав Python.

---

## 5. Критерии приёмки

Программный продукт считается прошедшим тестирование, если:

- все unit-тесты выполняются без ошибок;
- фактические результаты совпадают с ожидаемыми;
- отсутствуют исключения и аварийные завершения программы при корректных входных данных.

---

## 6. Ожидаемые результаты

Ожидается получение следующих результатов тестирования:

- успешное прохождение всех unit-тестов;
- подтверждение корректности вычислений площади и периметра фигур;
- отсутствие выявленных дефектов в реализованных функциях;
- отчёт о результатах тестирования, предоставляемый системой `unittest`.
